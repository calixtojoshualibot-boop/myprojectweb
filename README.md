# Vintage Cap Showcase System

A web application for showcasing and managing vintage NBA team caps with an HTML frontend and Node.js backend.

## Features

- **Showcase Display**: Beautiful responsive grid display of vintage NBA caps
- **Backend API**: Node.js Express server with RESTful API
- **Admin CRUD**: Full Create, Read, Update, Delete functionality
- **Cap Details**: Manage cap name, quality, and year released
- **Smooth Integration**: Frontend HTML connects seamlessly to backend JavaScript

## Installation

1. Install dependencies:
   ```
   npm install
   ```

2. Start the server:
   ```
   npm start
   ```

3. Open your browser and go to `http://localhost:3000`

## Features

### Main Showcase (`/`)
- Display vintage NBA caps in a beautiful grid layout
- Show cap name, quality, and year released
- Direct link to admin panel

### Admin Panel (`/admin`)
- Add new vintage caps
- Edit existing cap details
- Delete caps from collection
- Real-time updates reflected on showcase

## Development

For development with auto-restart:
```
npm run dev
```

## Technology Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend**: Node.js, Express.js
- **API**: RESTful endpoints for CRUD operations
